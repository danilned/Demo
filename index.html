Memory Game

Test yout memory by repeating colors

How to play

  1. Нажмите кнопку "Start", чтобы начать игру.
  2. Игра начнется с мигания одной кнопки. Запомните её цвет и нажмите ту же кнопку.
  3. С каждым раундом последовательность будет увеличиваться на один цвет. Запоминайте и повторяйте её.
  4. Если вы ошибетесь при повторении последовательности, игра закончится.
  5. Продолжайте, пока не достигнете максимальной длины последовательности или пока не проиграете.

Game rules

  • Игра начинается с одной кнопки, которая загорается и издает звук.
  • Вы должны точно повторить последовательность, которую показывает игра.
  • С каждым раундом добавляется новый цвет в последовательность.
  • Если вы ошибаетесь при повторении, игра завершается, и ваш счёт обнуляется.
  • Цель игры – повторять последовательности как можно дольше.
