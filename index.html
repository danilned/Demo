Memory Game

Test yout memory by repeating colors

How to play

  1. Начните игру, выбрав игрока, который ходит первым.
  2. Каждый игрок по очереди выполняет ход, следуя установленным правилам.
  3. Во время своего хода игрок может использовать доступные действия или карты (если применимо).
  4. Цель каждого игрока – набрать максимальное количество очков или выполнить игровую задачу.
  5. Игра продолжается до тех пор, пока один из игроков не достигнет условий победы.

Game rules

  • Каждый игрок должен по очереди выполнять свои ходы.
  • Игроки могут взаимодействовать друг с другом только в пределах своих ходов.
  • Выигрывает тот, кто первым наберет 10 очков.
  • Если игрок нарушает правило, он пропускает следующий ход.
  • Игра продолжается до тех пор, пока не будет объявлен победитель.
